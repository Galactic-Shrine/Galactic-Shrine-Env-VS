 [FR] Si la valeur est "true", les commentaires sont extraits du texte analysé et ajoutés à la structure de données.
S'il vaut false, il ignorera tous les commentaires lors de l'analyse et ne les stockera pas, ce qui permettra d'économiser de la mémoire et des allocations.
[EN] If set to true, comments are extracted from the parsed text and added to the data structure.

bool LancerDesExceptionsEnCasDerreur = true [get, set]
 [FR] Si true, l'instance Ini lèvera une exception si une erreur est trouvée.
Si false, l'analyseur s'arrête et renvoie une valeur nulle.
[EN] If true, the Ini instance will raise an exception if an error is found.

bool SauterLesLignesInvalides = false [get, set]
 [FR] S'il vaut true, il continue d'analyser le fichier même si une ligne mal formée est trouvée,
mais ne compte pas comme une erreur trouvée (c'est-à-dire que Ini.A_DesErreurs renverra false).
S'il a la valeur false, il lèvera une exception ou suivra une erreur, selon la valeur de LancerDesExceptionsEnCasDerreur,
lorsque l'analyseur rencontrera une ligne mal formée.
[EN] If set to true, it continues to analyze the file even if a malformed line is found,
but does not count as an error found (i.e. Ini.A_DesErrors will return false).
If set to false, it will throw an exception or follow an error, depending on the value of LancerDesExceptionsEnCasDerreur,
when the parser encounters a malformed line.
bool HabillagePropriete = true [get, set]
 [FR] S'il est défini sur true, il coupera l'espace blanc de la propriété lors de l'analyse.
S'il vaut false, il considérera tous les espaces de la ligne comme faisant partie de la propriété lors de l'extraction de la clé et des valeurs.
[EN] If set to true, it will cut the white space of the property during analysis.
If set to false, it will consider all spaces on the line as part of the property when extracting the key and values.
bool HabillageSections = true [get, set]
 [FR] Si la valeur est "true", le nom de la section sera coupé des espaces blancs lors de l'analyse.

[EN] If set to "true", the section name will be cut from the white spaces when parsed.

bool HabillageCommentaires = true [get, set]
 [FR] Si la valeur est "true", les espaces blancs seront supprimés des commentaires lors de l'analyse.

bool AutoriserLesSectionsDupliquees = false [get, set]
 [FR] Si la valeur est fixée à false et que le GalacticShrine.Configuration.Ini trouve une section dupliquée,
l'analyseur s'arrêtera avec une erreur. Si la valeur est fixée à true, les sections dupliquées sont autorisées dans le fichier,
mais seul un élément GalacticShrine.Modele.Configuration.Ini.Section sera créé dans la collection GalacticShrine.Configuration.DonneesIni.Sections.
[EN] If the value is set to false and the GalacticShrine.Configuration.Ini finds a duplicated section, the parser will stop with an error. If the value is set to true, duplicated sections are allowed in the file,
but only a GalacticShrine.Modele.Configuration.Ini.Section element will be created in the GalacticShrine.Configuration.DonneesIni.Sections collection.
bool AutoriserLesClesSansSection = true [get, set]
 [FR] Permet d'avoir des clés au début du fichier, avant qu'une section ne soit définie.

[EN] Allows you to have keys at the beginning of the file, before a section is defined.
These keys do not belong to any section and are stored in the special field GalacticShrine.Configuration.DonneesIni.ProprieteGlobales.
If set to false and the ini file contains keys outside a section, the parser will stop with an error.
